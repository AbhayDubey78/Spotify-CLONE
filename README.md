# 🎧 Spotify Clone (Ad-Free Music Player)

A sleek and responsive **Spotify-inspired music player** built using **HTML, CSS, and JavaScript** that allows users to play songs seamlessly — **without any ads interruption**.


## 📌 Features

* 🎵 **Ad-Free Music Playback** 🔥
* ⏯️ Play / Pause / Next / Previous controls
* 📂 Dynamic song loading from local directory
* 🎨 Clean and responsive UI
* 📊 Progress bar with real-time updates
* 🖼️ Album art & song info display
* ⚡ Smooth and fast performance (no backend needed)

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla)
